Hover

Detects when the player's mouse (or touch/gamepad cursor) is hovering over tagged instances in the 3D world, using a raycast from the camera.

Hoverable instances are marked using a CollectionService tag (configurable via Config.Tag, default "Hoverable").

INFO

This module is client-only. Requiring it on the server returns an empty table.

local hover = Hover.new()
hover:Start()

hover.HoveringStarted:Connect(function(instance)
	print("Started hovering:", instance:GetFullName())
end)

hover.Clicked:Connect(function(instance)
	print("Clicked while hovering:", instance:GetFullName())
end)

Returns the instance currently being hovered, or nil if nothing is hovered.

Init​

</>
Hover:Init() → ()

Sets up tag observation so tagged instances become hoverable. Called automatically by Hover:Start if not already initialized — you generally don't need to call this yourself.

Start​

</>
Hover:Start() → ()

Begins listening for mouse/touch/gamepad hovers over tagged instances. Calls Hover:Init automatically if it hasn't run yet.

Stop​

</>
Hover:Stop() → ()

Stops listening for hovers and clears any current hover state, firing Hover.HoveringEnded if something was being hovered.

Destroy​

</>
Hover:Destroy() → ()

Stops the instance, disconnects all internal listeners, and destroys its signals. The Hover instance should not be used after calling this.
